# MAD PRACTICAL & VIVA `24-11-2025`

This project demonstrates a simple Android application with the following features:
- Opening a second activity and passing data.
- Applying a tween animation to an ImageView.
- Changing the background color of the main layout via an AlertDialog.

## Features

*   **Main Activity (MainActivity.kt)**:
    *   Displays a TextView, an ImageView, and three buttons.
    *   Passes text from `MainActivity` to `SecondActivity`.
    *   Applies a tween animation (`tween_anim.xml`) to an `ImageView`.
    *   Allows changing the background color of the `LinearLayout` through a color selection dialog.

*   **Second Activity (SecondActivity.kt)**:
    *   Receives and displays text passed from `MainActivity`.
    *   Displays an image (`uvpce_logo.png`).
